## Required output files

Create the following PDFs before final submission:

- reports/phase1_report.pdf
- reports/phase2_report.pdf
- reports/final_report.pdf

## Recommended flow

Recommended flow:
1. Draft report content from markdown notes in this repository.
2. Export each document to PDF using your preferred editor.
3. Keep final PDF names exactly as required above.

## Word target

Final report word target: 2500-3000 words (excluding figures/tables/references).
